# Этап 6. Наложение отметки

Обзор проекта: [golosuy_web_service_f33e701e.plan.md](golosuy_web_service_f33e701e.plan.md)

## Задачи

- Пользователь выбирает номер варианта из списка обнаруженных квадратиков
- Тип отметки: галочка ✓ или крестик ✗ (рандом)
- Рендер в `[src/lib/render/drawMark.ts](../../src/lib/render/drawMark.ts)`:
  - Рисование на canvas в координатах **оригинального** (не выровненного) изображения
  - Стиль: чёрные линии, толщина пропорциональна размеру квадратика, лёгкий случайный наклон (±3°) для естественности
  - Размер отметки: 70–85% площади квадратика
- Превью до/после с возможностью сменить вариант
